# ColorMyWorld - 绘梦世界 - v1.1.3

**AI 驱动的个性化涂色书生成平台**

## 版本信息
- **当前版本**: v1.1.3
- **更新日期**: 2026-06-01

## 项目简介
ColorMyWorld 是一个基于 Web 的 AI 创意平台，专为儿童和家长设计。通过前沿的 AI 技术，将想象力转化为可打印的高质量涂色书。

## 技术栈
- **前端框架**: Next.js 14.2.35 (App Router)
- **UI 库**: React 18.3.1
- **样式**: Tailwind CSS v4 + PostCSS 8.5.6
- **语言**: TypeScript 5.5.3
- **AI SDK**: @google/genai (最新版)
- **动画**: motion/react 12.34.3
- **图标**: lucide-react 0.575.0
- **PDF**: jsPDF 2.5.1

## 核心特性
- 🎨 5 种艺术风格（幼儿、标准、精细、卡通、写实）
- 🌍 21 种语言支持
- 🤖 6 种 AI 引擎可选（Gemini、OpenAI、DeepSeek、Claude、Doubao、Qianwen）
- 📄 PDF 导出功能
- 💬 智能聊天助手
- 🎭 丰富的动画和交互效果
