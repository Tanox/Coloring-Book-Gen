# 功能规范：绘梦世界 (ColorMyWorld) - v1.3.0

## 1. 魔法生成流 (Magic Flow)

### 1.1 主题输入
- 支持自然语言描述（如"穿着宇航服在火星采蘑菇的猫"）。
- 自动关联孩子姓名，在封面页显式展示。

### 1.2 多维度风格选择
- **幼儿 (Simple)**：极简粗线条，适合蜡笔涂抹。
- **标准 (Standard)**：平衡的线条密度。
- **精细 (Detailed)**：曼陀罗风格，适合彩铅。
- **卡通 (Cartoon)**：更具角色个性的构图。
- **写实 (Realistic)**：学术速写风格。
- **国际化**：所有风格的名称和描述均已完全国际化，支持 21 种语言。

### 1.3 故事生成模式
- 可选功能，开启后为每一页生成一段短小的、连贯的背景故事。

### 1.4 图像参数精调 (Image Generation Controls)
- **画面比例 (Aspect Ratio)**：支持 1:1, 3:4, 4:3, 9:16, 16:9。
- **图像尺寸 (Resolution)**：Gemini 专属功能，支持 1K, 2K, 4K 分辨率。
- **艺术风格 (Art Style)**：5 种预设风格（Simple/Standard/Detailed/Cartoon/Realistic）。

### 1.5 并发生成
- **并发控制**：支持多页图像并发生成，支持 2 个并发任务。
- **进度指示**：实时显示生成进度（已生成页数/总页数）。
- **可取消**：支持在生成过程中查看已完成的页面。

## 2. 设置中心 (Settings Center)
- **全局偏好**：允许用户设置默认的 AI 引擎、艺术风格、分辨率和纵横比。
- **持久化存储**：所有设置均保存在本地 LocalStorage 中，跨会话保留。
- **表单同步**：生成表单会自动从设置中心拉取默认值，减少重复操作。
- **语言切换**：提供 21 种语言的即时切换功能。
- **API 密钥验证**：实时验证 API 密钥配置状态，提供视觉反馈。

## 3. 预览和导出

### 3.1 PDF 导出
- **分页渲染**：A4 纵向/横向构图，自动根据图像比例优化。
- **封面页**：包含主题和孩子姓名的封面。
- **故事文本**：在故事模式开启时，每页包含生成的故事文本。

### 3.2 单页重绘
- **即时修正**：在预览图库中，用户可对不满意的单页发起重新生成。
- **状态同步**：重绘成功后，页面自动更新，确保数据一致性。

### 3.3 图像懒加载
- **懒加载**：使用 Intersection Observer 实现图像懒加载，提升首屏性能。
- **加载占位**：使用 shimmer 动画作为加载占位，提升感知体验。

## 4. 智能助手 (Chat)
- **魔法小助手**：常驻右下角，提供灵感推荐与功能引导。
- **对话历史**：保持当前会话的对话历史上下文。
- **多语言**：自动使用当前选择的语言与用户交流。

## 5. 多 AI 引擎支持
- **支持引擎**：Gemini（图像+故事+聊天）、OpenAI（图像+故事+聊天）、DeepSeek（故事+聊天）、Claude（故事+聊天）、Doubao（故事+聊天）、Qianwen（故事+聊天）。
- **动态能力**：根据所选 AI 引擎，自动启用/禁用相应功能选项。
- **密钥配置**：支持通过环境变量（NEXT_PUBLIC_*_API_KEY）配置 API 密钥。
