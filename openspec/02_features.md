# 功能规范：绘梦世界 (ColorMyWorld) - v0.5.14

## 1. 魔法生成流 (Magic Flow)
### 1.1 主题输入
- 支持自然语言描述（如“穿着宇航服在火星采蘑菇的猫”）。
- 自动关联孩子姓名，在封面页显式展示。

### 1.2 多维度风格选择
- **幼儿 (Simple)**：极简粗线条，适合蜡笔涂抹。
- **标准 (Standard)**：平衡的线条密度。
- **精细 (Detailed)**：曼陀罗风格，适合彩铅。
- **卡通 (Cartoon)**：更具角色个性的构图。
- **写实 (Realistic)**：学术速写风格。
- **国际化**：所有风格的名称和描述均已完全国际化，支持11种语言。

### 1.3 故事生成模式
- 可选功能，开启后为除封面外的每一页生成一段短小的、具有连贯性的背景故事。

## 2. 交互式画布 (Interactive Canvas)
- **涂色技术**：采用 `multiply` 混合模式，确保涂色不遮挡黑白底图的线条。
- **工具集**：13 色彩虹调色板、2-60px 动态笔刷、一键重置功能。
- **交互性能**：支持多点触控（平板友好）与鼠标操作，笔刷响应延迟控制在 16ms 以内。

## 3. 内容交付与数据管理
### 3.1 PDF 导出
- **分页渲染**：A4 纵向构图，自动优化分页。
- **多语言修复**：自动嵌入 Noto Sans SC 字体，彻底解决中文字符乱码问题。

### 3.2 单页重绘
- **即时修正**：在预览图库中，用户可对不满意的单页发起重新生成。
- **状态同步**：重绘成功后，页面和**当前操作的**历史记录将自动更新，确保数据一致性。

### 3.3 分享功能
- **整本分享**：通过 Web Share API 分享书的主题和基本信息链接。
- **单页分享**：支持将生成的黑白线条图作为文件直接通过社交软件分享。
- **剪贴板兼容**：在不支持 Web Share 的浏览器中自动退化为链接复制。

### 3.4 配置迁移
- **导入/导出**：支持将 API 密钥及偏好设置导出为 JSON 文件，方便在不同设备间迁移。

### 3.5 历史管理
- **自动保存**：生成成功后立即入库 IndexedDB。
- **版本预览**：支持从历史记录中直接重新加载整本书的配置与图像。

## 4. 智能助手 (Chat)
- 魔法小助手角色，常驻右下角，提供灵感推荐与功能引导。
