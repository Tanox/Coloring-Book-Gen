# 项目愿景：绘梦世界 (ColorMyWorld) - v0.5.14

## 1. 项目定义
**绘梦世界** 是一款基于 Web 的 AI 创意平台，旨在将前沿的人工智能技术转化为简单、有趣且富有教育意义的儿童创作工具。

## 2. 核心使命
通过 AI 赋能，让每个孩子都能轻松创作出专属于自己的、高质量的、可实物化的艺术作品。

## 3. 核心价值
- **个性化优先**：AI 不只是生成内容，而是生成与孩子建立情感连接的内容（如姓名嵌入、兴趣主题）。
- **极简体验**：消除技术门槛，让 3 岁以上的孩子在家长辅助下即可完成创作。
- **即刻交付**：从创意爆发到实物打印（PDF）的链路极短。

## 4. 目标受众
- **核心用户**：3-10 岁的儿童。
- **主要客户**：注重创意启蒙与亲子互动的家长、艺术教育工作者。

## 5. 项目准则
- **隐私第一**：所有敏感配置（API 密钥）及生成内容仅保留在本地。
- **公平性**：支持多种 AI 引擎，不绑定单一供应商，提供平滑的回退体验。
- **可访问性**：针对儿童视觉习惯优化的 UI 布局。
