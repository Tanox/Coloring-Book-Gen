# 更新日志 (Changelog)

## v0.5.0
### 基础
- **里程碑发布**：正式进入 v0.5.0 阶段。
- **全局版本同步**：更新所有核心组件与配置文件版本号，确保开发环境与线上版本高度一致。

## v0.4.4
### 优化
- **代码清理**：废弃并移除了过时的 `app/services/geminiService.ts`，由更完善的 `aiService.ts` 统一接管所有 AI 服务调度。
- **一致性增强**：同步更新所有核心文件的头部注释版本号至 v0.4.4，确保项目颗粒度对齐。

## v0.4.3
### 修复
- **PDF 乱码修复**：优化了 `pdfService.ts` 中的字体加载与嵌入逻辑，强制每一页重新应用中文字体，彻底解决导出 PDF 时出现的“鬼画符”问题。

### 新增
- **配置导入/导出**：在设置面板新增“数据管理”板块，支持将所有已配置的 API 密钥导出为 `.json` 文件，并支持一键导入。

### 优化
- **UI 增强**：设置面板增加动画与视觉分区，提升复杂设置项的易读性。

## v0.4.2
### 优化
- **UI 风格革命**：全面移除 `shadow-` 相关样式，转向“扁平拟物”风格，通过边框和色彩层级构建 UI，视觉更加清爽。
- **文档深度拆分**：将 `project.md` 拆分为 `project.md` (愿景) 与 `features.md` (功能)，并同步更新 `architecture.md`、`uiux.md`、`agents.md`，实现颗粒度对齐。
- **版本标记**：同步 `metadata.json` 与 `App.tsx` 的版本号至 v0.4.2。

## v0.4.1
### 文档
- **全量汉化**：将所有规范文档切换为专业中文，便于团队理解。
- **架构对齐**：完善 `architecture.md`，记录了 IndexedDB 结构。

## v0.4.0
### 新增
- **多供应商连接测试**：在设置面板中新增实时 API 校验逻辑。
- **豆包 (Doubao) 支持**：集成火山引擎 API。

## v0.3.9
### 新增
- **高级图像引擎**：首次引入 DALL-E 3 支持。
- **故事助手**：引入 Claude 3.5 Sonnet 生成高质量故事。

## v0.2.6
### 基础
- **初始版本**：基于 Google Gemini 的核心涂色生成流程发布。
